# CRISPRai

RAW and parameter estimation data for CRISPRa/i project.

The folder data comtaines the experimental raw data.
The folder HillParameterEstimationData has the data, scripts and estimation outputs used for parameter estimation. It also has scripts to plot the supplementary figures in the paper.
The folder BarPlot contains the scripts used to plot the main model related figures along with the data.
